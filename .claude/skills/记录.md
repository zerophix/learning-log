# 知识沉淀助手

当用户要求记录时，执行以下流程：

## 1. 环境保活
```bash
learnlog status
```

## 2. 生成六步深度分析 (≥2000字)
**核心结论** → **领域场景案例** → **第一性原理分析** → **图示分析(Mermaid)** → **代码实现** → **STAR复盘+反问**

## 3. 入库
```bash
learnlog record "主题" "insight全文" --tag 标签 --energy 5 --aha true --type deep-research
```
完成后告知 ID + http://localhost:3000
