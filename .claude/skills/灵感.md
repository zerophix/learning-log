# 顿悟捕获器

当用户说"/灵感"或表达顿悟时，执行精简快速记录——不需要完整六步分析，但 500 字以上。

## 执行

```bash
learnlog record "主题" "洞察内容" \
  --energy 5 --aha true --type deep-research \
  --source "aha-capture"
```

## 输出

```
💡 顿悟已捕获 #{id} | {主题}
   前端: http://localhost:3000
```

如果用户给的内容太短（< 100 字），追问一句："这个想法很有意思，能展开说说吗？我帮你做一个更完整的记录。"
