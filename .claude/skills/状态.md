# 学习日志状态速查

当用户输入 `/状态`、`/status`、`/zt` 或问"学习状态如何"时，执行：

```bash
learnlog status
learnlog feed -n 5
```

输出格式：
```
📊 学习日志状态
   记录: 28 条 | 标签: 63 个 | 关联: 37 条

🏷️ 最近沉淀:
   #28 [06/26 22:08] 系统设计核心原则 ⚡⚡⚡⚡⚡💡
   #27 [06/26 22:07] 跨项目 AI 协作的实践验证 ⚡⚡⚡⚡
   ...

💡 /记录 — 记录新洞察  |  http://localhost:3000 — 查看时间线
```

如果后端未运行，自动 `learnlog start`。
