# 可用命令
```
📝 /记录    深度知识沉淀（协议对齐格式：结论→Mermaid+ASCII→Why→架构→决策→STAR）
📊 /状态    系统状态速查
💡 /灵感    快速顿悟捕获
🔧 /服务    后台服务管理（安装/状态/重启/日志）
❓ /命令    这条消息
```

## 服务管理
```bash
learnlog service install     # 安装持久化服务（开机自启）
learnlog service status      # 查看后台服务状态
learnlog service restart     # 重启服务
learnlog service logs        # 查看运行日志
learnlog service uninstall   # 卸载服务
```

## 记录维护
```bash
# 长文推荐用管道（避免 shell 截断）
echo "完整洞察..." | learnlog record "主题" --pipe

# 更新任意字段
learnlog update <id> --energy 5 --aha true
echo "修正后全文..." | learnlog update <id> --pipe

# 删除记录
learnlog delete <id>         # 确认后删除
learnlog delete <id> -f      # 跳过确认
```
